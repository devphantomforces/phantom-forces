1. **Use an alt account** — Never run combat scripts on your main account. A dedicated alt protects your primary profile from any ban risk.

4. **Toggle off ESP when spectated** — Some server moderators spectate suspicious players. Disabling obvious visual modules when you suspect scrutiny reduces your profile significantly.
5. **Monitor the Discord** — The phantom forces script Discord posts real-time alerts when specific modules become detectable or when a Phantom Forces update affects the script's stealth layers.

## 🐛 Common Issues & Fixes

### ❌ Silent Aim Not Connecting Shots
**Cause:** FOV radius set too narrow, or the target is outside the aim lock range.
**Fix:** Open the GUI and increase the Silent Aim FOV slider to 35–45°. Ensure "Nearest Enemy" targeting mode is selected rather than "Crosshair Target."

---

### ❌ No Recoil Only Works on Some Guns
**Cause:** Running an outdated script version that doesn't include recoil profiles for newer weapons.
**Fix:** Update to the latest release. As of v6.3.0, all 127 currently available Phantom Forces weapons have recoil compensation profiles built in.

---

### ❌ Inf Credits Not Accumulating
**Cause:** Phantom Forces pushed a server-side economy validation update that blocked the previous credit hook.
**Fix:** Download the latest script version. Inf credits patches are always prioritized in hotfix releases and noted in the Changelog.

---

### ❌ GUI Doesn't Appear After Execution
**Cause:** Script executed before the match fully loaded, or executor level is insufficient.
**Fix:** Wait until you're fully in a live match (not the lobby screen), then re-execute. Ensure your executor is Level 7 or above.

---

### ❌ Anti-Kick Not Preventing Server Ejections
**Cause:** The server is running a newer anti-exploit version that patched the previous bypass method.
**Fix:** Check for a script update. Anti-kick is one of the fastest-patched modules after Phantom Forces updates — the latest version will include the corrected bypass.

---

### ❌ Ragebot Targeting Teammates
**Cause:** Team filter not enabled in the ragebot settings.
**Fix:** In the GUI, open the Ragebot section and enable **"Enemies Only"** mode. This restricts targeting to opposing team members exclusively.

---

### ❌ Script Errors on Execute in Solara
**Cause:** Solara-specific API differences with the standard build.
**Fix:** Use the dedicated Solara build available in the Releases tab. The phantom forces silent aim script Solara variant is specifically compiled and tested for Solara's execution environment.
